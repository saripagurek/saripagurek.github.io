Predicate Logic Calculator Recursive Python algorithm to parse and evaluate given predicate logic expressions and return a completed truth table, interfaced with React.js using brython.
Use Calculator
The calculator takes in a boolean logic expression of the format:
((a|b)&(!c))
and catches errors when brackets are missing or invalid characters are inputted.
